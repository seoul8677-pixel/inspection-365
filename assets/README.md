# assets/

이 폴더에는 사이트에서 참조하는 정적 이미지가 들어갑니다. 아래 파일들은 아직 없으며 추후 추가 필요.

## 추가 필요 파일

| 파일명 | 권장 사양 | 용도 |
|---|---|---|
| `og-image.png` | 1200×630, PNG | 카카오톡·페이스북·트위터 공유 시 썸네일 (Open Graph / Twitter Card) |
| `favicon.png` | 32×32 또는 64×64, PNG | 브라우저 탭 아이콘 |
| `apple-touch-icon.png` | 180×180, PNG | iOS 홈 화면 추가 시 아이콘 |

> 파일을 추가하면 `index.html`의 `<head>`에 이미 참조 경로가 걸려 있으므로 별도 코드 수정은 필요 없습니다.
